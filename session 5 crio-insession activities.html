A media query is a CSS technique used to apply styles based on the device’s screen size, resolution, or other properties. It helps make websites responsive by adjusting layouts for different screen sizes. body { background-color: white; } @media (max-width: 600px) { body { background-color: lightgray; } } @media (min-width: 1024px) { p { font-size: 20px; } } @media (max-width: 768px) { .desktop-only { display: none; } } CSS Grid is a layout system that allows you to create responsive and flexible grid-based designs. It works by defining rows and columns in a container and placing elements accordingly. .grid-container { display: grid; grid-template-columns: repeat(3, 1fr); /* 3 equal columns */ grid-template-rows: auto; /* Rows adjust automatically */ gap: 10px; /* Space between grid items */ } .grid-container { display: grid; grid-template-columns: 1fr 1fr; grid-template-rows: 100px 100px; gap: 10px; } .grid-container { display: grid; grid-template-columns: 200px 1fr 2fr; }
